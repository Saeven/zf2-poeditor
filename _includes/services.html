So what's the deal?

Install

Install gettext, then install this with composer, and add this module to your application config.

Translate

Access /translate to use a web-based GUI to parse and translate all your PHP and Twig code.

Compile

Save and compile PO/MO files automagically, in 1 simple step, from right within your app.

Enjoy

Configs and PO files are automatically module specific, PLUS, it's hella easy to use.
